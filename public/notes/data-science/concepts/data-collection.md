# Data Collection

## 概念简介

Data Collection（数据收集）是获取用于分析的原始数据的过程。数据质量直接影响后续分析的有效性。

## 数据来源

常见的数据来源包括：

- **数据库** - 企业内部的结构化数据
- **API** - 第三方服务提供的数据接口
- **网络爬虫** - 从网站抓取公开数据
- **传感器** - IoT设备产生的实时数据
- **调查问卷** - 通过问卷收集的一手数据

## 与其他概念的关系

数据收集是基于 [[problem-framing]] 确定的需求进行的。

收集到的原始数据通常需要通过 [[data-cleaning]] 进行清洗，才能用于分析。

数据收集的质量会影响 [[feature-engineering]] 的效果和最终 [[model-selection]] 的性能。
